## 请求约定



其中可以携带 学校名 参数的一定要带上, 可以优化查询请求



```json
{
    // 存放单个信息, 多用于条件查询
	"info": {
        "info1":"",
        "info2":"",
        "info3":"",
        "info4":""
    },
    // 对象列表, 多用于批量添加
    "infoList":[
        {
          // 对象信息1
        },{
          // 对象信息2
        },{
          // 对象信息3
        }
    ],
    // id 列表 多用于批量删除
    "idList":[
        id_1,
        id_2,
        id_3,
        id_4,
        id_5
    ]
}
```



tip: 了解一下 mysql 数据库的 增删改查 语句的编写, 可以帮助理解
